\# 🌾 Farm2Fuel

A digital marketplace for agricultural waste.



\## Problem

Farmers burn crop stubble causing pollution.

No platform exists to sell agri-waste.



\## Solution

Farm2Fuel connects farmers with bioenergy 

companies to buy straw, husk \& stubble.



\## Tech Stack

\- Frontend: HTML, CSS, JavaScript

\- Backend: Python + Flask

\- Database: SQLite



\## How to Run

pip install flask flask-cors

python app.py

Open http://localhost:5050
